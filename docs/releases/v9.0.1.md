# NOOP v9.0.1

A polish-and-translation release on top of 9.0.0. The headline is **full German, French and Spanish** across the whole app, **pull-to-sync** on the Today screen, and a stack of visual fixes and reliability improvements. Everything still runs on your own device, offline, no account. Update from the Releases page, AltStore, or `brew upgrade`.

## New

- **German, French and Spanish, everywhere (#453).** The complete Android UI moved onto translated string resources, the Apple String Catalogs were finished, and coverage is now structurally complete for German, French and Spanish across iPhone, Mac, Android, the watch app and complications. A CI gate keeps future copy from silently drifting back to English-only.
- **Pull to sync on Today (#334).** Pull down on Today to request a fresh strap-history offload — iPhone, Mac and Android. It reuses the same safe manual-sync path as the "Sync now" button, so it only fires when the strap is connected, bonded and not already syncing; the sync status chip owns the longer progress.
- **The day-cycle sky behind your cards, on by default.** The background sky now extends behind the whole Today scroll out of the box (previously opt-in). It stays a Settings toggle, so anyone who preferred the flat canvas can turn it back off; existing choices are kept.
- **Trend charts show the inspected date (#492).** Tapping or scrubbing a point on an Android trend chart now prefixes the date, matching the Apple readout.

## Fixed

- **macOS could no longer be granted Bluetooth (#429).** Recent macOS refuses to bind a Bluetooth permission to an unsigned app bundle, so the strap read "disconnected" and even a permission reset wouldn't stick. The macOS build is now ad-hoc signed, giving it a code identity the system can grant to.
- **WHOOP 5.0/MG battery % didn't show on iPhone/Mac (#490).** The strap's battery characteristic was read once before the secure link was up (and rejected), then never retried — so the percentage stayed blank. It's now re-read after bonding and kept current, matching Android.
- **Activity-file (FIT) imports didn't fill in steps (#483).** A FIT file for a foot sport (walk/run/hike) now imports its step total into the day, when the day has no other step source.
- **A batch of Today polish (#486, #492).** The day title no longer clips its bottom edge, Strain drops a stray "%" (it's a 0–21/0–100 score, not a percentage), the source badges tuck into their cards instead of floating above, the workout Source labels fit their column, and the Compare "Add metric" action no longer wraps one letter per line.
- **A duplicate resting-stress buzz (#330).** The older experimental stress nudge is removed in favour of the newer Stress check-in, so a single HRV dip can't trigger two haptic paths.
- **Android reconnect could stall next to the official WHOOP app (#313).** A strap the phone still holds connected (co-resident with the WHOOP app) is now kept on fast direct reconnect instead of a passive mode that stalls on it — keeping the battery poll and history sync alive.
